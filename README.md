# Dot net api 
proxy web api in .netcore to allow vue app to search images from flickr api.
## .NET core must be installed.
Go to root folder and run below command in command line
"dotnet run"
## NodeJS and NPM must be installed.
"..VueApp/ImageExplorer" run:
1. npm install
2. npm run serve
## Browser
http://localhost:8080/#
